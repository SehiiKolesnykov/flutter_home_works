# hw_6

Створіть додаток "Мій профіль", що містить три основні екрани:
1. Головний екран (HomeScreen): Містить вітальний текст і кнопку для переходу
   до екрану профілю.
2. Екран профілю (ProfileScreen): Відображає інформацію про користувача (ім'я,
   вік, місце роботи) та кнопку для переходу на екран "Контакти".
3. Екран контактів (ContactsScreen): Містить список контактів з іменами та
   номерами телефонів.
   Вимоги:
1. Навігація:
   ○ Використовуйте іменовані маршрути для переходу між екранами.
   ○ Використовуйте Navigator.pushNamed() для переходу вперед і
   Navigator.pop() для повернення.
2. Розташування елементів:
   ○ На Головному екрані розмістіть текст у центрі екрана з вітанням,
   використовуючи Column.
   ○ На Екрані профілю:
   ■ Використовуйте Row для розташування основної інформації
   (наприклад, зображення профілю та текст з інформацією про
   користувача).
   ■ Додайте кнопку з текстом "Контакти", розташувавши її внизу
   екрану.
3. Інтервали та відступи:
   ○ Використовуйте Padding і SizedBox для налаштування відступів між
   елементами.
   ○ Елементи повинні не “злипатись”.
4. Додатково:
   ○ На Екрані контактів реалізуйте простий список з іменами та номерами
   телефонів за допомогою Column.
   ○ Також за бажанням додайте трохи стилізації: змініть колір кнопок або тло
   екрану тощо.
   Приклад:
   ● Головний екран: Текст "Ласкаво просимо!" та кнопка "Перейти до профілю".
   ● Екран профілю: Зображення профілю (можна взяти будь-який контейнер
   замість зображення), ім'я та вік користувача, кнопка "Контакти".
   ● Екран контактів: Прості текстові блоки з іменем та телефоном, розділені
   SizedBox.
