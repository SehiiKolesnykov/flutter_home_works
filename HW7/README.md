# hw_7

Створіть додаток, який відображає профіль користувача з окремими виджетами
для різних частин інтерфейсу.

Екран профілю:
● Створіть простий інтерфейс профілю із заголовком та розділами (Divider) для опису та статистики.
Компоненти профілю:
● Профільний заголовок: Відображає аватарку користувача, його ім'я та короткий опис, наприклад,
професію.
● Статистика користувача: Відображає кілька статистичних показників, таких як кількість проектів,
підписників та підписок.
Користувацькі віджети:
● Створіть окремий виджет для профільного заголовка (наприклад, ProfileHeader).
● Створіть окремий виджет для статистики користувача (наприклад, UserStatusWidget), де кожен
параметр статистики передається через конструктор.
Опис функціоналу

1. Головний екран:
   ○ Використовуйте Scaffold з AppBar, який відображатиме назву екрана, наприклад, “User Profile”.
   ○ Вміст екрана повинен бути в Column і містити користувальнікі віджети ProfileHeader та
   UserStatusWidget.
2. Користувацькі віджети:
   ○ ProfileHeader: Виджет для відображення аватарки користувача (використовуйте CircleAvatar),
   імені (наприклад, "Oleksandr") та короткого опису ("Flutter Developer").
   ○ UserStats: Виджет для відображення показників: проектів, підписників і підписок. Кожен показник
   створіть через функцію чи окремий метод (наприклад, _buildStatistic). Кожен показник – це текст
   із відповідним кількістю.
3. Адаптивність:
   ○ Налаштуйте розміри тексту та відступи так, щоб вигляд був зручним як на маленьких, так і на
   великих екранах (на планшеті). (використовуйте LayoutBuilder або Flutter пакети на слайді №44)
