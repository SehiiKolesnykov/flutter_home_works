# hw_8

Потрібно створити простий, але функціональний додаток "To-Do List" (Список завдань),
який дозволяє користувачам додавати, видаляти та виконувати завдання. Завдання
повинно включати анімації для додавання нових завдань, виконання та видалення
існуючих завдань, а також взаємодію з користувачем через кнопки та текстові поля.

Функціонал:

Список завдань (Lists):

● Використовуйте ListView, щоб відобразити перелік завдань, який може
прокручуватися, якщо завдань багато.
● Додавайте нове завдання у верхній частині списку, щоб воно одразу було видно.
Текстове поле (Text Field):
● Додайте TextField для введення тексту нового завдання.
● Якщо текстове поле порожнє, кнопка додавання завдання має бути неактивною.
Кнопки (Buttons):
● Додати завдання: Кнопка поруч із текстовим полем, яка додає нове завдання
до списку, коли поле не порожнє.
● Позначити як виконане: Кнопка або чекбокс у кожному завданні, що відзначає
його як виконане. Виконані завдання повинні виглядати по-іншому (наприклад,
сірим або з перекресленим текстом).
● Видалити завдання: Кнопка для видалення завдання зі списку.

Анімації (Animations):

● Додавання завдання: Використовуйте анімацію при додаванні нового
завдання. Наприклад, можна зробити ефект плавного з’явлення (fade-in).
● Позначення завдання як виконаного: Використовуйте анімацію для зміни
стану завдання з "не виконано" на "виконано". Це може бути зміна кольору або
повільне перекреслення тексту.
● Видалення завдання: Використовуйте анімацію видалення, наприклад, плавне
зникнення (fade-out) або рух завдання за межі екрану перед видаленням.
Додатковий функціонал (не обов'язково, але можна погратись :)))) ):
● Додайте можливість переміщувати завдання в списку (drag-and-drop).
● Використовуйте Hero анімацію для перемикання між екраном списку завдань і
екраном деталізації (екран деталізації може містити повний опис завдання та
інші деталі).

Поради:

● Використовуйте AnimatedList для списку завдань, щоб додавання та
видалення анімувалися автоматично.
● Можете використати SlideTransition, FadeTransition, або
ScaleTransition для анімації додавання/видалення.
● Приділіть увагу інтерфейсу користувача, щоб він був зручним і зрозумілим.
